## String.format()

```java
Scanner scanner = new Scanner(System.in);
while (scanner.hasNextInt()) {
    int i = scanner.nextInt();
    String s = String.format("%04d", i);
    System.out.println(s);
}
```

## DecimalFormat()

```java
Scanner scanner = new Scanner(System.in);
while (scanner.hasNextInt()) {
    int i = scanner.nextInt();
    DecimalFormat df = new DecimalFormat("0000");
    System.out.println(df.format(i));
}
```
